# JanSevak-Copilot
JanSevak Copilot is an agentic AI system built on Amazon Bedrock (Claude 3.5 &amp; Llama 3) to automate last-mile governance. It empowers 3M+ frontline workers with vernacular voice-to-action, autonomous form-filling, and policy reasoning to bridge the service delivery gap for 900M+ citizens. Built for Bharat-scale impact.
